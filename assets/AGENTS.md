# AGENTS.md

## 1. 目标

`assets` 存放非代码资产，例如 Bevy 场景资源、图片、字体、音频、模型、像素风素材、示例教学素材等。

## 2. 目标实现的路径

- 按资源类型和使用场景组织资产。
- 所有资产应记录来源、授权和用途。
- Bevy 使用的模型、贴图、字体和音频应保持可加载、可压缩、可替换。
- 教学素材应尽量与具体课程插件解耦。

## 3. 需要联网查找/参考的资料与核心思想

需要查找：

- Bevy asset pipeline 文档。
- glTF、PNG、WebP、OGG、字体格式资料。
- 开源素材授权协议，如 CC0、CC-BY、MIT、Apache-2.0。
- 像素风素材制作和压缩资料。

核心思想：

- 资产必须有明确来源和授权。
- 渲染资产服务于教学场景，不应绑定具体业务逻辑。
- 优先使用轻量、跨平台、可缓存的格式。

## 4. 不允许做什么事情

- 不允许提交来源不明或授权不清的素材。
- 不允许提交过大的二进制资产而不说明用途。
- 不允许在资产目录中存放密钥、用户数据或生成缓存。
- 不允许让资产文件名包含敏感信息。
